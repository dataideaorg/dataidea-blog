# Welcome to Your First Blog Post

This is your first blog post! You can edit this file to add your own content.

## Getting Started

To create a new blog post, simply create a new markdown file in the `docs/blog/posts/` directory.

## Writing Posts

Blog posts support:
- **Bold text** and *italic text*
- [Links](https://example.com)
- Code blocks:

```python
def hello_world():
    print("Hello, World!")
```

- And much more!

## Next Steps

2. Create new posts in the `docs/blog/posts/` directory
3. Run `mkdocs serve` to preview your blog locally
4. Deploy to your hosting platform
